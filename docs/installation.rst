============
Installation
============

Requirements
------------

- Python 3.10 or later
- Cppcheck (for generating dump files)

From Source
-----------

.. code-block:: bash

   git clone https://github.com/user/cppcheckdata-shims.git
   cd cppcheckdata-shims
   pip install -e ".[dev,docs]"

From PyPI (when published)
--------------------------

.. code-block:: bash

   pip install cppcheckdata-shims

Verify Installation
-------------------

.. code-block:: bash

   casl --version
   casl --help
